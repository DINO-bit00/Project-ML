#  Hoax News Classification using Machine Learning

This project focuses on detecting hoax (fake) news using Machine Learning techniques.  
The model is trained to classify news articles into hoax or non-hoax categories based on text features.

---

##  Project Structure

- Naive_Baiyes.py  
- Implementasi_model.py
- preprocessing.py
- requirements.txt
- datasetberita.xlsx
- datasetberita.csv


---

## Dataset

The dataset used in this project was obtained from **Kaggle**.

- Dataset Name: *Deteksi Berita Hoaks Indo - Dataset*
- Source: https://www.kaggle.com/datasets/mochamadabdulazis/deteksi-berita-hoaks-indo-dataset

The dataset contains news articles labeled as hoax or factual.  
Preprocessing steps include text cleaning, tokenization, and feature extraction.
